## Setup

### Install dependencies

```sh
yarn install
```

### Create env file

```sh
cp .env.example .env
```

### Runs the app in development mode.

```sh
yarn start
```

Now you can open http://localhost:3000 to view it in the browser.

### Builds the app for production to the `dist` folder

```sh
yarn build
```

### Runs the app in production mode.

```sh
yarn serve
```
